 # Run Locally


### Install/Build dependencies

```shell
npm rebuild
```

### If this error 'esbuild' is not recognized as an internal or external command,operable program or batch file.
```shell
npm run esbuild --bundle src/main.js --outfile=www/main.js --servedir=www
```
### Start the app

```shell
npm run dev
```
